# FraudWatch Quality Standards

## Content

- Accurate
- Clear
- Up to date
- Plain language
- Reviewed before publication

## Software

- Responsive design
- No broken links
- No JavaScript errors
- Accessible navigation
- Consistent styling

## User Experience

- Users can find help in three clicks or fewer.
- Important actions are easy to identify.
- Disclaimers are clear and visible.

## Security

- Protect user privacy.
- Minimize stored personal data.
- Moderate public reports.
