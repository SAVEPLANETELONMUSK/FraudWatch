# FraudWatch Ethics

## We Will

- Respect user privacy.
- Provide educational guidance.
- Encourage independent verification.
- Distinguish facts from user reports.
- Promote responsible reporting.

## We Will Not

- Guarantee fund recovery.
- Publish unverified accusations.
- Sell user information.
- Encourage harassment or retaliation.

## Commitment

FraudWatch exists to help people make safer decisions and access trustworthy guidance after fraud.
