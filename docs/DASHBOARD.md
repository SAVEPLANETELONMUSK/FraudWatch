# FraudWatch Dashboard

## Purpose

The Dashboard is the main entry point to FraudWatch.

It should help visitors quickly find the right action.

## Primary Actions

- Verify Something
- Report a Scam
- Get Recovery Guidance
- Read Scam Alerts
- Browse Scam Guides
- Contact Support

## Future Widgets

- Latest Scam Alerts
- Recently Updated Articles
- Featured Safety Tip
- Official Resources
- Search Bar
