# FraudWatch Product Specification

## Mission

Help people prevent fraud, verify suspicious activity, report scams, recover safely, and access trusted educational and emotional support resources.

---

## Public Features

- Home
- Verify
- Report Fraud
- Recovery Centre
- Support Centre
- Education Centre
- Contact

---

## Registered User Features

- User account
- Saved searches
- Saved reports
- Recovery checklist
- Notifications

---

## Moderator Features

- Review reports
- Approve or reject reports
- Remove abusive submissions
- Publish verified alerts

---

## Future Features

- AI scam analysis
- Browser extension
- Mobile apps
- Business portal
- Global scam intelligence

---

## Core Principles

- Transparency
- Privacy
- Security
- Accuracy
- Compassion
- Education

Never guarantee fund recovery.

Never declare someone fraudulent without evidence.

Always encourage independent verification.
